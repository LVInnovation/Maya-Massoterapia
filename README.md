# ✨ Mayà Massoterapia & Estética

Sistema de agendamento desenvolvido para clínicas de estética, massoterapia e bem-estar.

O objetivo do projeto é oferecer uma experiência simples, elegante e totalmente responsiva para clientes realizarem agendamentos e para administradores gerenciarem profissionais, serviços e horários.

---

## 🌟 Principais Recursos

### 👩‍⚕️ Gestão de Profissionais

* Cadastro de profissionais
* Edição de informações
* Exclusão de profissionais
* Organização dos atendimentos

### 💆 Gestão de Serviços

* Cadastro de serviços
* Configuração de duração
* Edição e exclusão
* Associação aos profissionais

### 📅 Sistema de Agendamento

* Escolha de profissional
* Escolha de serviço
* Seleção de data e horário
* Controle de conflitos de agenda
* Cancelamento de agendamentos
* Visualização dos horários disponíveis

### 🔐 Área Administrativa

* Login administrativo local
* Gerenciamento completo do sistema
* Controle dos cadastros

### 💾 Armazenamento Local

* Funcionamento 100% em LocalStorage
* Não depende de servidor
* Não depende de banco de dados externo
* Persistência dos dados no navegador
